Instructions for Running the TicTacToe Client:
 - Make sure you have Java Runtime Environment installed!
 - http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html
 - All players must follow these steps

1. Extract files from "TTTClient.jar"
	- Open cmd 
	- Navigate to folder with .jar file in it
	- Type "jar xf TTTClient.jar" without quotations and press enter

2. Ensure server is turned on
	- Send and email to sherryn18@asmsa.org with the subject line "Server ON" without quotation marks

3. Start TTTClient
	- Double click run.bat to automatically compile and run the game.
